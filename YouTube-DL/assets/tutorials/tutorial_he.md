# 🎓 מדריך — Ultimate YouTube Downloader

## 🚀 הקדמה
ברוכים הבאים למדריך השימוש באפליקציה! אפליקציה זו מאפשרת להוריד סרטונים ורשימות השמעה מ-YouTube בפורמט וידאו או אודיו, עם תמיכה ברשימות השמעה פרטיות.

האפליקציה מציעה שני מצבים עיקריים:

- 🔥 **הורדה פשוטה**: לניהול סרטונים אחד אחרי השני (או רשימות השמעה)
- 📋 **הורדה באצווה (Batch)**: להורדת מספר כתובות URL בבת אחת

------

## 🔥 הורדה פשוטה

1. **העתק את כתובת ה-URL** של סרטון YouTube 🔗

2. **הדבק אותה** בשדה ה-URL

3. **לחץ** על הכפתור "➕ אימות"

4. **המתן** שמידע הסרטון ייטען (תמונה ממוזערת, משך, גודל)

   **בחר** את האפשרויות שלך:

   - 🎥 **וידאו**: בחר רזולוציה (Best, 2160p, 1440p, 1080p, 720p, 480p, 360p, 240p, 144p)
   - 🎵 **אודיו בלבד**: בחר קצב סיביות (Best, 320, 256, 192, 128, 96, 64, 32 kbps) ופורמט (M4A או MP3)

   **לחץ** על "⬇️ הורדה"

   **בחר** את תיקיית היעד

### טיפים 💡

- ניתן **להוסיף מספר סרטונים** לתור ההמתנה לפני תחילת ההורדה
- **הגודל הכולל** ומספר הסרטונים מוצגים על כפתור ההורדה
- השתמש בכפתור **🗑️ ריקון התור** כדי למחוק את כל הסרטונים בהמתנה
- לחץ על הכפתור **ℹ️** ליד כל סרטון כדי לראות מידע מפורט (פורמטים זמינים, תיאור, וכו')
- לחץ על **❌** כדי להסיר סרטון מתור ההמתנה

## 📚 רשימות השמעה

### רשימות השמעה ציבוריות

פשוט העתק את כתובת ה-URL של רשימת ההשמעה בלשונית "הורדה בודדת", האפליקציה תזהה אוטומטית את כל הסרטונים! ניתן לשנות את אפשרויות הוידאו/אודיו לפני תחילת ההורדה.

האפליקציה:

- ✅ טוענת אוטומטית את כל הסרטונים ברשימת ההשמעה
- ✅ מציגה תמונה ממוזערת ומידע לכל סרטון
- ✅ מאפשרת להתאים אישית את האפשרויות לכל סרטון בנפרד
- ✅ מורידה הכל בפעולה אחת

### 🪪 רשימות השמעה פרטיות
#### שיטה 1: התחברות אוטומטית דרך הדפדפן (מומלץ ⭐)

**האפליקציה משתמשת אוטומטית ב-cookies שלך מ-Firefox אם אתה מחובר ל-YouTube!**

1. **התחבר** לחשבון ה-YouTube שלך ב-Firefox
2. **העתק** את כתובת ה-URL של רשימת ההשמעה הפרטית שלך
3. **הדבק אותה** באפליקציה
4. ✅ **עובד אוטומטית!** האפליקציה ניגשת ל-cookies שלך מ-Firefox

> 💡 **טיפ**: הישאר מחובר ל-YouTube ב-Firefox כדי שהאפליקציה תוכל תמיד לגשת לרשימות ההשמעה הפרטיות שלך ללא טיפול נוסף.

#### שיטה 2: ייצוא ידני של cookies (חלופה)

אם השיטה האוטומטית לא עובדת או אם אתה משתמש בדפדפן אחר, עליך לספק **cookies של YouTube**:

1. התקן תוסף דפדפן:
   - **Firefox**: [cookies.txt](https://addons.mozilla.org/he/firefox/addon/cookies-txt/) או [get cookies](https://addons.mozilla.org/he/firefox/addon/get_cookies/)
   - **Chrome**: [Get cookies.txt LOCALLY](https://chrome.google.com/webstore)

2. התחבר ל-YouTube
3. **ייצא** את ה-cookies שלך בפורמט Netscape (קובץ `.txt`)
4. **לחץ** על הכפתור "🪪 ⬆️ cookies.txt" בפינה השמאלית העליונה של האפליקציה
5. **בחר** את קובץ ה-`cookies.txt` שלך
6. **בדוק** עם רשימת ההשמעה הפרטית שלך

> ⚠️ **אזהרה**: לעולם אל תשתף את קובץ ה-cookies.txt שלך, הוא מכיל את פרטי ההתחברות שלך!

### ⚠️ עצות
- אל תמחק את קובץ ה-cookies.txt
- אם מופיעה שגיאה, טען מחדש את ה-cookies

## 📦 הורדה באצווה (Batch)

הלשונית "הורדה באצווה" מאפשרת להוריד מספר סרטונים עם **אותן הגדרות** לכולם.

### שימוש

1. **פתח** את הלשונית "הורדה באצווה"
2. **הזן** כתובות URL של הסרטונים (אחת בכל שורה) באזור הטקסט
   - או לחץ על "⬆️ טעינה מקובץ" כדי לייבא קובץ `.txt` המכיל את כתובות ה-URL שלך
3. **בחר** את הסוג:
   - 🎥 **וידאו**: עם רזולוציה משותפת לכולם
   - 🎵 **אודיו בלבד**: עם קצב סיביות משותף לכולם
4. **בחר** את הרזולוציה (אם וידאו) ואת קצב הסיביות של האודיו
5. **לחץ** על "⬇️ הורדה"
6. **בחר** את תיקיית היעד

### ההבדל מההורדה הפשוטה

| הורדה פשוטה | הורדה באצווה |
| --- | --- |
| אפשרויות **מותאמות אישית** לכל סרטון | אפשרויות **זהות** לכולם |
| מציג תמונות ממוזערות ומידע מפורט | ממשק מפושט |
| אידיאלי למספר קטן של סרטונים מגוונים | אידיאלי להרבה סרטונים דומים |

## ⚙️ אפשרויות מתקדמות

### רזולוציית וידאו

| רזולוציה | שימוש מומלץ | גודל משוער (שעה אחת) |
| --- | --- | --- |
| **Best** | איכות טובה ביותר זמינה | משתנה |
| **2160p (4K)** | מסכי 4K, ארכוב | ~4-8 GB |
| **1440p (2K)** | צגים ברזולוציה גבוהה | ~2-4 GB |
| **1080p (Full HD)** | שימוש סטנדרטי, איזון מיטבי ⭐ | ~1-2 GB |
| **720p (HD)** | מכשירים ניידים, חיסכון במקום | ~500 MB-1 GB |
| **480p** | חיבור איטי, אחסון מוגבל | ~300-500 MB |
| **360p** | חיבור איטי מאוד | ~200-300 MB |

> 💡 **עצה**: לשימוש יומיומי, **1080p** מציעה את האיזון הטוב ביותר בין איכות לגודל.

### קצב סיביות אודיו

| קצב סיביות | איכות | שימוש מומלץ | גודל (שעה אחת) |
| --- | --- | --- | --- |
| **Best** | מקסימום זמין | ארכוב, חובבי אודיו ⭐ | משתנה |
| **320 kbps** | מעולה | מוזיקה באיכות גבוהה | ~140 MB |
| **256 kbps** | טובה מאוד | שימוש סטנדרטי | ~115 MB |
| **192 kbps** | טובה | איזון איכות/גודל | ~85 MB |
| **128 kbps** | סבירה | פודקאסטים, הרצאות | ~60 MB |
| **96 kbps** | מקובלת | קול בלבד | ~45 MB |
| **64 kbps** | נמוכה | חיבור איטי מאוד | ~30 MB |

### פורמט אודיו

- **M4A**:
  - ✅ איכות טובה יותר באותו גודל
  - ✅ קבצים קלים יותר
  - ✅ פורמט YouTube מקורי (ללא המרה)
  - ❌ תאימות פחותה לנגנים ישנים
- **MP3**:
  - ✅ תואם לכל המכשירים
  - ✅ נתמך באופן נרחב
  - ✅ ניתן להתאמה אישית (קצב סיביות לפי הבחירה)
  - ❌ דורש המרה (נדרש FFmpeg)

## ❓ בעיות נפוצות

### "The playlist does not exist"

**גורמים אפשריים:**

1. רשימת ההשמעה **פרטית** → ודא שאתה מחובר ל-YouTube ב-Firefox, או ספק קובץ `cookies.txt`
2. כתובת ה-URL שגויה → בדוק שהעתקת את כתובת ה-URL המלאה של רשימת ההשמעה
3. רשימת ההשמעה נמחקה → בדוק שהיא עדיין קיימת ב-YouTube

### "ERROR: unable to download video data"

**גורמים אפשריים:**

1. חיבור אינטרנט לא יציב
2. סרטון נמחק או פרטי
3. YouTube שינה את הפורמט שלו → עדכן את yt-dlp: `pip install -U yt-dlp`

### הורדה איטית

- **פתרונות:**
  - ✅ בדוק את חיבור האינטרנט שלך 📶
  - ✅ YouTube מגביל לפעמים את המהירות לפי המיקום שלך
  - ✅ נסה להוריד בזמן אחר
  - ✅ הורד ברזולוציה נמוכה יותר (720p במקום 1080p)

### שגיאת המרה ל-MP3

**שגיאה**: `ERROR: ffmpeg not found`

**פתרון**: ההמרה ל-MP3 דורשת התקנת **FFmpeg** במערכת שלך.

**התקנת FFmpeg:**

```bash
# Ubuntu/Debian
sudo apt install ffmpeg

# Fedora
sudo dnf install ffmpeg

# Arch Linux
sudo pacman -S ffmpeg

# macOS (Homebrew)
brew install ffmpeg

# Windows (Chocolatey)
choco install ffmpeg
```

**בדיקת ההתקנה:**

```bash
ffmpeg -version
```
