# 🎓 Οδηγός — Ultimate YouTube Downloader

## 🚀 Παρουσίαση
Καλώς ήρθατε στον οδηγό χρήσης της εφαρμογής! Αυτή η εφαρμογή σας επιτρέπει να κατεβάζετε βίντεο και λίστες αναπαραγωγής από το YouTube σε μορφή βίντεο ή ήχου, με υποστήριξη ιδιωτικών λιστών αναπαραγωγής.

Η εφαρμογή προσφέρει δύο κύριες λειτουργίες:

- 🔥 **Απλή λήψη**: για διαχείριση βίντεο ένα προς ένα (ή λίστες αναπαραγωγής)
- 📋 **Λήψη μαζική (Batch)**: για λήψη πολλών URL ταυτόχρονα

------

## 🔥 Απλή λήψη

1. **Αντιγράψτε τη διεύθυνση URL** του βίντεο YouTube 🔗

2. **Επικολλήστε την** στο πεδίο URL

3. **Κάντε κλικ** στο κουμπί "➕ Επαλήθευση"

4. **Περιμένετε** να φορτωθούν οι πληροφορίες του βίντεο (μικρογραφία, διάρκεια, μέγεθος)

   **Επιλέξτε** τις επιλογές σας:

   - 🎥 **Βίντεο**: επιλέξτε την ανάλυση (Best, 2160p, 1440p, 1080p, 720p, 480p, 360p, 240p, 144p)
   - 🎵 **Μόνο ήχος**: επιλέξτε το bitrate (Best, 320, 256, 192, 128, 96, 64, 32 kbps) και τη μορφή (M4A ή MP3)

   **Κάντε κλικ** στο "⬇️ Λήψη"

   **Επιλέξτε** τον φάκελο προορισμού

### Συμβουλές 💡

- Μπορείτε να **προσθέσετε πολλά βίντεο** στην ουρά αναμονής πριν ξεκινήσετε τη λήψη
- Το **συνολικό μέγεθος** και ο αριθμός των βίντεο εμφανίζονται στο κουμπί λήψης
- Χρησιμοποιήστε το κουμπί **🗑️ Εκκαθάριση ουράς** για διαγραφή όλων των βίντεο σε αναμονή
- Κάντε κλικ στο κουμπί **ℹ️** δίπλα σε κάθε βίντεο για να δείτε λεπτομερείς πληροφορίες (διαθέσιμες μορφές, περιγραφή, κ.λπ.)
- Κάντε κλικ στο **❌** για αφαίρεση ενός βίντεο από την ουρά αναμονής

## 📚 Λίστες αναπαραγωγής

### Δημόσιες λίστες αναπαραγωγής

Απλά αντιγράψτε τη διεύθυνση URL της λίστας αναπαραγωγής στην καρτέλα "Μονή λήψη", η εφαρμογή θα εντοπίσει αυτόματα όλα τα βίντεο! Μπορείτε να τροποποιήσετε τις επιλογές βίντεο/ήχου πριν ξεκινήσετε τη λήψη.

Η εφαρμογή:

- ✅ Φορτώνει αυτόματα όλα τα βίντεο της λίστας αναπαραγωγής
- ✅ Εμφανίζει μικρογραφία και πληροφορίες για κάθε βίντεο
- ✅ Επιτρέπει την προσαρμογή των επιλογών για κάθε βίντεο ξεχωριστά
- ✅ Κατεβάζει τα πάντα σε μία λειτουργία

### 🪪 Ιδιωτικές λίστες αναπαραγωγής
#### Μέθοδος 1: Αυτόματη σύνδεση μέσω προγράμματος περιήγησης (Συνιστάται ⭐)

**Η εφαρμογή χρησιμοποιεί αυτόματα τα cookies σας από το Firefox εάν είστε συνδεδεμένοι στο YouTube!**

1. **Συνδεθείτε** στον λογαριασμό σας στο YouTube μέσω Firefox
2. **Αντιγράψτε** τη διεύθυνση URL της ιδιωτικής λίστας αναπαραγωγής σας
3. **Επικολλήστε την** στην εφαρμογή
4. ✅ **Λειτουργεί αυτόματα!** Η εφαρμογή έχει πρόσβαση στα cookies σας από το Firefox

> 💡 **Συμβουλή**: Παραμείνετε συνδεδεμένοι στο YouTube μέσω Firefox ώστε η εφαρμογή να έχει πάντα πρόσβαση στις ιδιωτικές λίστες αναπαραγωγής σας χωρίς πρόσθετους χειρισμούς.

#### Μέθοδος 2: Χειροκίνητη εξαγωγή cookies (Εναλλακτική)

Εάν η αυτόματη μέθοδος δεν λειτουργεί ή εάν χρησιμοποιείτε άλλο πρόγραμμα περιήγησης, πρέπει να παρέχετε τα **cookies του YouTube**:

1. Εγκαταστήστε μια επέκταση προγράμματος περιήγησης:
   - **Firefox**: [cookies.txt](https://addons.mozilla.org/el/firefox/addon/cookies-txt/) ή [get cookies](https://addons.mozilla.org/el/firefox/addon/get_cookies/)
   - **Chrome**: [Get cookies.txt LOCALLY](https://chrome.google.com/webstore)

2. Συνδεθείτε στο YouTube
3. **Εξάγετε** τα cookies σας σε μορφή Netscape (αρχείο `.txt`)
4. **Κάντε κλικ** στο κουμπί "🪪 ⬆️ cookies.txt" πάνω αριστερά στην εφαρμογή
5. **Επιλέξτε** το αρχείο `cookies.txt` σας
6. **Δοκιμάστε** με την ιδιωτική λίστα αναπαραγωγής σας

> ⚠️ **Προσοχή**: Μην μοιράζεστε ποτέ το αρχείο cookies.txt, περιέχει τα διαπιστευτήρια σύνδεσής σας!

### ⚠️ Συμβουλές
- Μην διαγράφετε το αρχείο cookies.txt
- Εάν εμφανιστεί σφάλμα, επαναφορτώστε τα cookies

## 📦 Μαζική λήψη (Batch)

Η καρτέλα "Μαζική λήψη" σας επιτρέπει να κατεβάσετε πολλά βίντεο με τις **ίδιες ρυθμίσεις** για όλα.

### Χρήση

1. **Ανοίξτε** την καρτέλα "Μαζική λήψη"
2. **Εισάγετε** τις διευθύνσεις URL των βίντεο (μία ανά γραμμή) στην περιοχή κειμένου
   - Ή κάντε κλικ στο "⬆️ Φόρτωση από αρχείο" για εισαγωγή αρχείου `.txt` που περιέχει τις διευθύνσεις URL σας
3. **Επιλέξτε** τον τύπο:
   - 🎥 **Βίντεο**: με κοινή ανάλυση για όλα
   - 🎵 **Μόνο ήχος**: με κοινό bitrate για όλα
4. **Επιλέξτε** την ανάλυση (εάν είναι βίντεο) και το bitrate ήχου
5. **Κάντε κλικ** στο "⬇️ Λήψη"
6. **Επιλέξτε** τον φάκελο προορισμού

### Διαφορά από την απλή λήψη

| Απλή λήψη | Μαζική λήψη |
| --- | --- |
| Επιλογές **προσαρμοσμένες** ανά βίντεο | Επιλογές **πανομοιότυπες** για όλα |
| Εμφανίζει μικρογραφίες και λεπτομερείς πληροφορίες | Απλοποιημένη διεπαφή |
| Ιδανική για λίγα ποικίλα βίντεο | Ιδανική για πολλά παρόμοια βίντεο |

## ⚙️ Προηγμένες επιλογές

### Ανάλυση βίντεο

| Ανάλυση | Συνιστώμενη χρήση | Κατά προσέγγιση μέγεθος (1ω) |
| --- | --- | --- |
| **Best** | Καλύτερη διαθέσιμη ποιότητα | Μεταβλητό |
| **2160p (4K)** | Οθόνες 4K, αρχειοθέτηση | ~4-8 GB |
| **1440p (2K)** | Οθόνες υψηλής ευκρίνειας | ~2-4 GB |
| **1080p (Full HD)** | Τυπική χρήση, καλύτερη ισορροπία ⭐ | ~1-2 GB |
| **720p (HD)** | Φορητές συσκευές, εξοικονόμηση χώρου | ~500 MB-1 GB |
| **480p** | Αργή σύνδεση, περιορισμένος χώρος | ~300-500 MB |
| **360p** | Πολύ αργή σύνδεση | ~200-300 MB |

> 💡 **Συμβουλή**: Για καθημερινή χρήση, το **1080p** προσφέρει την καλύτερη ισορροπία ποιότητας/μεγέθους.

### Bitrate ήχου

| Bitrate | Ποιότητα | Συνιστώμενη χρήση | Μέγεθος (1ω) |
| --- | --- | --- | --- |
| **Best** | Μέγιστη διαθέσιμη | Αρχειοθέτηση, audiophiles ⭐ | Μεταβλητό |
| **320 kbps** | Εξαιρετική | Μουσική υψηλής ποιότητας | ~140 MB |
| **256 kbps** | Πολύ καλή | Τυπική χρήση | ~115 MB |
| **192 kbps** | Καλή | Ισορροπία ποιότητας/μεγέθους | ~85 MB |
| **128 kbps** | Αποδεκτή | Podcasts, διαλέξεις | ~60 MB |
| **96 kbps** | Ικανοποιητική | Μόνο φωνή | ~45 MB |
| **64 kbps** | Χαμηλή | Πολύ αργή σύνδεση | ~30 MB |

### Μορφή ήχου

- **M4A**:
  - ✅ Καλύτερη ποιότητα σε ίσο μέγεθος
  - ✅ Ελαφρύτερα αρχεία
  - ✅ Εγγενής μορφή YouTube (χωρίς μετατροπή)
  - ❌ Λιγότερο συμβατή με παλαιά προγράμματα αναπαραγωγής
- **MP3**:
  - ✅ Συμβατό με όλες τις συσκευές
  - ✅ Ευρέως υποστηριζόμενο
  - ✅ Προσαρμόσιμο (bitrate της επιλογής σας)
  - ❌ Απαιτεί μετατροπή (απαιτείται FFmpeg)

## ❓ Συχνά προβλήματα

### "The playlist does not exist"

**Πιθανές αιτίες:**

1. Η λίστα αναπαραγωγής είναι **ιδιωτική** → Βεβαιωθείτε ότι είστε συνδεδεμένοι στο YouTube μέσω Firefox, ή παρέχετε ένα αρχείο `cookies.txt`
2. Η διεύθυνση URL είναι λανθασμένη → Ελέγξτε ότι αντιγράψατε την πλήρη διεύθυνση URL της λίστας αναπαραγωγής
3. Η λίστα αναπαραγωγής διαγράφηκε → Ελέγξτε ότι εξακολουθεί να υπάρχει στο YouTube

### "ERROR: unable to download video data"

**Πιθανές αιτίες:**

1. Ασταθής σύνδεση στο διαδίκτυο
2. Το βίντεο διαγράφηκε ή είναι ιδιωτικό
3. Το YouTube άλλαξε τη μορφή του → Ενημερώστε το yt-dlp: `pip install -U yt-dlp`

### Αργή λήψη

- **Λύσεις:**
  - ✅ Ελέγξτε τη σύνδεσή σας στο διαδίκτυο 📶
  - ✅ Το YouTube περιορίζει μερικές φορές την ταχύτητα ανάλογα με την τοποθεσία σας
  - ✅ Δοκιμάστε να κατεβάσετε σε άλλη ώρα
  - ✅ Κατεβάστε με χαμηλότερη ανάλυση (720p αντί για 1080p)

### Σφάλμα μετατροπής MP3

**Σφάλμα**: `ERROR: ffmpeg not found`

**Λύση**: Η μετατροπή MP3 απαιτεί την εγκατάσταση του **FFmpeg** στο σύστημά σας.

**Εγκατάσταση FFmpeg:**

```bash
# Ubuntu/Debian
sudo apt install ffmpeg

# Fedora
sudo dnf install ffmpeg

# Arch Linux
sudo pacman -S ffmpeg

# macOS (Homebrew)
brew install ffmpeg

# Windows (Chocolatey)
choco install ffmpeg
```

**Έλεγχος εγκατάστασης:**

```bash
ffmpeg -version
```

### "Permission denied" κατά τη λήψη

**Πιθανές αιτίες:**

1. Ο φάκελος προορισμού είναι προστατευμένος από εγγραφή
2. Ένα αρχείο με το ίδιο όνομα είναι ήδη ανοιχτό
3. Το antivirus σας μπλοκάρει την εγγραφή

**Λύσεις:**

- ✅ Επιλέξτε έναν φάκελο στον προσωπικό σας κατάλογο (Έγγραφα, Λήψεις)
- ✅ Κλείστε κάθε ανοιχτό αρχείο βίντεο
- ✅ Προσθέστε εξαίρεση στο antivirus σας
