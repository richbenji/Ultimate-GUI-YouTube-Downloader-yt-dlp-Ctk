# 🎓 チュートリアル — Ultimate YouTube Downloader

## 🚀 はじめに
アプリケーションの使用ガイドへようこそ!このアプリケーションでは、YouTubeの動画とプレイリストを動画または音声形式でダウンロードでき、プライベートプレイリストにも対応しています。

アプリケーションには2つの主要なモードがあります:

- 🔥 **シンプルダウンロード**: 動画を1つずつ管理(またはプレイリスト)
- 📋 **バッチダウンロード**: 複数のURLを一度にダウンロード

------

## 🔥 シンプルダウンロード

1. YouTube動画の**URLをコピー** 🔗

2. URLフィールドに**貼り付け**

3. 「➕ 確認」ボタンを**クリック**

4. 動画情報(サムネイル、長さ、サイズ)が読み込まれるまで**待機**

   オプションを**選択**:

   - 🎥 **動画**: 解像度を選択(Best、2160p、1440p、1080p、720p、480p、360p、240p、144p)
   - 🎵 **音声のみ**: ビットレートを選択(Best、320、256、192、128、96、64、32 kbps)と形式(M4AまたはMP3)

   「⬇️ ダウンロード」を**クリック**

   保存先フォルダーを**選択**

### ヒント 💡

- ダウンロードを開始する前に、キューに**複数の動画を追加**できます
- **合計サイズ**と動画の数がダウンロードボタンに表示されます
- **🗑️ キューをクリア**ボタンを使用して、待機中のすべての動画を削除します
- 各動画の横にある**ℹ️**ボタンをクリックして、詳細情報(利用可能な形式、説明など)を表示します
- **❌**をクリックして、キューから動画を削除します

## 📚 プレイリスト

### 公開プレイリスト

「単一ダウンロード」タブにプレイリストのURLをコピーするだけで、アプリケーションがすべての動画を自動的に検出します!ダウンロードを開始する前に、動画/音声オプションを変更できます。

アプリケーションは:

- ✅ プレイリストのすべての動画を自動的に読み込みます
- ✅ 各動画のサムネイルと情報を表示します
- ✅ 各動画のオプションを個別にカスタマイズできます
- ✅ すべてを1回の操作でダウンロードします

### 🪪 プライベートプレイリスト
#### 方法1: ブラウザーを介した自動接続(推奨 ⭐)

**YouTubeにログインしている場合、アプリケーションはFirefoxのCookieを自動的に使用します!**

1. FirefoxでYouTubeアカウントに**ログイン**
2. プライベートプレイリストのURLを**コピー**
3. アプリケーションに**貼り付け**
4. ✅ **自動的に動作します!** アプリケーションがFirefoxのCookieにアクセスします

> 💡 **ヒント**: Firefoxで常にYouTubeにログインしたままにしておくと、追加操作なしでアプリケーションがプライベートプレイリストにアクセスできます。

#### 方法2: Cookieの手動エクスポート(代替)

自動方法が機能しない場合、または別のブラウザーを使用している場合は、**YouTube Cookie**を提供する必要があります:

1. ブラウザー拡張機能をインストール:
   - **Firefox**: [cookies.txt](https://addons.mozilla.org/ja/firefox/addon/cookies-txt/)または[get cookies](https://addons.mozilla.org/ja/firefox/addon/get_cookies/)
   - **Chrome**: [Get cookies.txt LOCALLY](https://chrome.google.com/webstore)

2. YouTubeにログイン
3. Netscape形式でCookieを**エクスポート**(`.txt`ファイル)
4. アプリケーションの左上にある「🪪 ⬆️ cookies.txt」ボタンを**クリック**
5. `cookies.txt`ファイルを**選択**
6. プライベートプレイリストで**テスト**

> ⚠️ **警告**: cookies.txtファイルを共有しないでください。ログイン資格情報が含まれています!

### ⚠️ アドバイス
- cookies.txtファイルを削除しないでください
- エラーが表示された場合は、Cookieを再読み込みしてください

## 📦 バッチダウンロード

「バッチダウンロード」タブでは、すべての動画に対して**同じ設定**で複数の動画をダウンロードできます。

### 使用方法

1. 「バッチダウンロード」タブを**開く**
2. テキストエリアに動画のURLを**入力**(1行に1つ)
   - または「⬆️ ファイルから読み込み」をクリックして、URLを含む`.txt`ファイルをインポート
3. タイプを**選択**:
   - 🎥 **動画**: すべてに共通の解像度
   - 🎵 **音声のみ**: すべてに共通のビットレート
4. 解像度(動画の場合)とオーディオビットレートを**選択**
5. 「⬇️ ダウンロード」を**クリック**
6. 保存先フォルダーを**選択**

### シンプルダウンロードとの違い

| シンプルダウンロード | バッチダウンロード |
| --- | --- |
| 動画ごとに**カスタマイズされた**オプション | すべてに**同じ**オプション |
| サムネイルと詳細情報を表示 | シンプルなインターフェース |
| 少数の多様な動画に最適 | 多数の類似動画に最適 |

## ⚙️ 高度なオプション

### 動画解像度

| 解像度 | 推奨用途 | 概算サイズ(1時間) |
| --- | --- | --- |
| **Best** | 利用可能な最高品質 | 可変 |
| **2160p (4K)** | 4K画面、アーカイブ | ~4-8 GB |
| **1440p (2K)** | 高解像度モニター | ~2-4 GB |
| **1080p (Full HD)** | 標準使用、最良のバランス ⭐ | ~1-2 GB |
| **720p (HD)** | モバイルデバイス、スペース節約 | ~500 MB-1 GB |
| **480p** | 低速接続、限られたストレージ | ~300-500 MB |
| **360p** | 非常に低速な接続 | ~200-300 MB |

> 💡 **アドバイス**: 日常使用には、**1080p**が品質とサイズの最良のバランスを提供します。

### オーディオビットレート

| ビットレート | 品質 | 推奨用途 | サイズ(1時間) |
| --- | --- | --- | --- |
| **Best** | 最大利用可能 | アーカイブ、オーディオファン ⭐ | 可変 |
| **320 kbps** | 優秀 | 高品質音楽 | ~140 MB |
| **256 kbps** | 非常に良い | 標準使用 | ~115 MB |
| **192 kbps** | 良い | 品質/サイズのバランス | ~85 MB |
| **128 kbps** | 許容範囲 | ポッドキャスト、講義 | ~60 MB |
| **96 kbps** | 受け入れ可能 | 音声のみ | ~45 MB |
| **64 kbps** | 低い | 非常に低速な接続 | ~30 MB |

### オーディオ形式

- **M4A**:
  - ✅ 同じサイズでより良い品質
  - ✅ より軽いファイル
  - ✅ YouTubeネイティブ形式(変換なし)
  - ❌ 古いプレーヤーとの互換性が低い
- **MP3**:
  - ✅ すべてのデバイスと互換性
  - ✅ 広くサポートされている
  - ✅ カスタマイズ可能(選択したビットレート)
  - ❌ 変換が必要(FFmpegが必要)

## ❓ よくある問題

### "The playlist does not exist"

**考えられる原因:**

1. プレイリストが**プライベート**である → FirefoxでYouTubeにログインしているか確認するか、`cookies.txt`ファイルを提供してください
2. URLが正しくない → プレイリストの完全なURLをコピーしたか確認してください
3. プレイリストが削除された → YouTubeにまだ存在するか確認してください

### "ERROR: unable to download video data"

**考えられる原因:**

1. インターネット接続が不安定
2. 動画が削除またはプライベート
3. YouTubeが形式を変更した → yt-dlpを更新: `pip install -U yt-dlp`

### ダウンロードが遅い

- **解決策:**
  - ✅ インターネット接続を確認 📶
  - ✅ YouTubeは場所に応じて速度を制限することがあります
  - ✅ 別の時間にダウンロードしてみてください
  - ✅ より低い解像度でダウンロード(1080pの代わりに720p)

### MP3変換エラー

**エラー**: `ERROR: ffmpeg not found`

**解決策**: MP3変換にはシステムに**FFmpeg**がインストールされている必要があります。

**FFmpegのインストール:**

```bash
# Ubuntu/Debian
sudo apt install ffmpeg

# Fedora
sudo dnf install ffmpeg

# Arch Linux
sudo pacman -S ffmpeg

# macOS (Homebrew)
brew install ffmpeg

# Windows (Chocolatey)
choco install ffmpeg
```

**インストールの確認:**

```bash
ffmpeg -version
```

### ダウンロードが99%で止まる

これは正常です!アプリケーションが動画と音声を**マージ**している(またはMP3に変換している)状態です。このステップには、次の要因に応じて数秒から数分かかることがあります:

- ファイルサイズ
- コンピューターの性能
- 選択した解像度

> 💡 **ヒント**: 進行状況バーが99%の間は、アプリケーションを閉じないでください!

### ダウンロード中に「Permission denied」

**考えられる原因:**

1. 保存先フォルダーが書き込み保護されている
2. 同じ名前のファイルが既に開いている
3. ウイルス対策ソフトウェアが書き込みをブロックしている

**解決策:**

- ✅ 個人ディレクトリ(ドキュメント、ダウンロード)内のフォルダーを選択
- ✅ 開いている動画ファイルを閉じる
- ✅ ウイルス対策ソフトウェアに例外を追加
