# 🎓 教程 — GOD (God Offers Downloads)

## 🚀 简介
欢迎使用应用程序使用指南!此应用程序允许您以视频或音频格式下载YouTube视频和播放列表,并支持私人播放列表。

应用程序提供两种主要模式:

- 🔥 **简单下载**: 逐个管理视频(或播放列表)
- 📋 **批量下载**: 一次下载多个URL

------

## 🔥 简单下载

1. **复制YouTube视频的URL** 🔗

2. **粘贴**到URL字段

3. **点击**"➕ 验证"按钮

4. **等待**视频信息加载(缩略图、时长、大小)

   **选择**您的选项:

   - 🎥 **视频**: 选择分辨率(Best、2160p、1440p、1080p、720p、480p、360p、240p、144p)
   - 🎵 **仅音频**: 选择比特率(Best、320、256、192、128、96、64、32 kbps)和格式(M4A或MP3)

   **点击**"⬇️ 下载"

   **选择**目标文件夹

### 提示 💡

- 您可以在开始下载之前**将多个视频添加**到队列
- **总大小**和视频数量显示在下载按钮上
- 使用**🗑️ 清空队列**按钮删除所有等待中的视频
- 点击每个视频旁边的**ℹ️**按钮查看详细信息(可用格式、描述等)
- 点击**❌**从队列中删除视频

------

## 📚 播放列表

### 公开播放列表

只需在"单次下载"选项卡中复制播放列表的URL,应用程序将自动检测所有视频!您可以在开始下载之前修改视频/音频选项。

应用程序:

- ✅ 自动加载播放列表中的所有视频
- ✅ 显示每个视频的缩略图和信息
- ✅ 允许单独自定义每个视频的选项
- ✅ 一次性下载所有内容

### 🪪 私人播放列表
#### 方法1: 通过浏览器自动连接(推荐 ⭐)

**如果您已登录YouTube,应用程序会自动使用Firefox的Cookie!**

1. 在Firefox中**登录**您的YouTube账户
2. **复制**您的私人播放列表的URL
3. **粘贴**到应用程序中
4. ✅ **自动工作!** 应用程序访问您在Firefox中的Cookie

> 💡 **提示**: 在Firefox中保持登录YouTube,以便应用程序始终可以访问您的私人播放列表,无需额外操作。

#### 方法2: 手动导出Cookie(替代方案)

如果自动方法不起作用或您使用其他浏览器,则需要提供**YouTube Cookie**:

1. 安装浏览器扩展:
   - **Firefox**: [cookies.txt](https://addons.mozilla.org/zh-CN/firefox/addon/cookies-txt/)或[get cookies](https://addons.mozilla.org/zh-CN/firefox/addon/get_cookies/)
   - **Chrome**: [Get cookies.txt LOCALLY](https://chrome.google.com/webstore)

2. 登录YouTube
3. 以Netscape格式**导出**您的Cookie(`.txt`文件)
4. **点击**应用程序左上角的"🪪 ⬆️ cookies.txt"按钮
5. **选择**您的`cookies.txt`文件
6. 使用您的私人播放列表**测试**

> ⚠️ **警告**: 切勿共享您的cookies.txt文件,它包含您的登录凭据!

### ⚠️ 建议
- 不要删除cookies.txt文件
- 如果出现错误,请重新加载Cookie

## 📦 批量下载

"批量下载"选项卡允许您使用**相同的设置**下载多个视频。

### 使用方法

1. **打开**"批量下载"选项卡
2. 在文本区域**输入**视频URL(每行一个)
   - 或点击"⬆️ 从文件加载"导入包含URL的`.txt`文件
3. **选择**类型:
   - 🎥 **视频**: 所有视频使用共同的分辨率
   - 🎵 **仅音频**: 所有视频使用共同的比特率
4. **选择**分辨率(如果是视频)和音频比特率
5. **点击**"⬇️ 下载"
6. **选择**目标文件夹

### 与简单下载的区别

| 简单下载 | 批量下载 |
| --- | --- |
| 每个视频的**自定义**选项 | 所有视频的**相同**选项 |
| 显示缩略图和详细信息 | 简化的界面 |
| 适合少量多样化视频 | 适合大量相似视频 |

## ⚙️ 高级选项

### 视频分辨率

| 分辨率 | 推荐用途 | 大约大小(1小时) |
| --- | --- | --- |
| **Best** | 最佳可用质量 | 可变 |
| **2160p (4K)** | 4K屏幕、存档 | ~4-8 GB |
| **1440p (2K)** | 高清显示器 | ~2-4 GB |
| **1080p (Full HD)** | 标准使用、最佳平衡 ⭐ | ~1-2 GB |
| **720p (HD)** | 移动设备、节省空间 | ~500 MB-1 GB |
| **480p** | 慢速连接、有限存储 | ~300-500 MB |
| **360p** | 非常慢的连接 | ~200-300 MB |

> 💡 **建议**: 对于日常使用,**1080p**提供了最佳的质量/大小平衡。

### 音频比特率

| 比特率 | 质量 | 推荐用途 | 大小(1小时) |
| --- | --- | --- | --- |
| **Best** | 最大可用 | 存档、音频发烧友 ⭐ | 可变 |
| **320 kbps** | 优秀 | 高质量音乐 | ~140 MB |
| **256 kbps** | 非常好 | 标准使用 | ~115 MB |
| **192 kbps** | 良好 | 质量/大小平衡 | ~85 MB |
| **128 kbps** | 可接受 | 播客、讲座 | ~60 MB |
| **96 kbps** | 尚可 | 仅语音 | ~45 MB |
| **64 kbps** | 较低 | 非常慢的连接 | ~30 MB |

### 音频格式

- **M4A**:
  - ✅ 相同大小下更好的质量
  - ✅ 更轻的文件
  - ✅ YouTube原生格式(无需转换)
  - ❌ 与旧播放器的兼容性较低
- **MP3**:
  - ✅ 与所有设备兼容
  - ✅ 广泛支持
  - ✅ 可定制(选择的比特率)
  - ❌ 需要转换(需要FFmpeg)

## ❓ 常见问题

### "The playlist does not exist"

**可能的原因:**

1. 播放列表是**私人的** → 确保您已在Firefox中登录YouTube,或提供`cookies.txt`文件
2. URL不正确 → 检查您是否复制了播放列表的完整URL
3. 播放列表已被删除 → 检查它是否仍然存在于YouTube上

### "ERROR: unable to download video data"

**可能的原因:**

1. 互联网连接不稳定
2. 视频已删除或私人
3. YouTube更改了其格式 → 更新yt-dlp: `pip install -U yt-dlp`

### 下载缓慢

- **解决方案:**
  - ✅ 检查您的互联网连接 📶
  - ✅ YouTube有时会根据您的位置限制速度
  - ✅ 尝试在其他时间下载
  - ✅ 以较低的分辨率下载(720p而不是1080p)

### MP3转换错误

**错误**: `ERROR: ffmpeg not found`

**解决方案**: MP3转换需要在您的系统上安装**FFmpeg**。

**安装FFmpeg:**

```bash
# Ubuntu/Debian
sudo apt install ffmpeg

# Fedora
sudo dnf install ffmpeg

# Arch Linux
sudo pacman -S ffmpeg

# macOS (Homebrew)
brew install ffmpeg

# Windows (Chocolatey)
choco install ffmpeg
```

**检查安装:**

```bash
ffmpeg -version
```

### 下载在99%处停止

这是正常的!应用程序正在**合并**视频和音频(或转换为MP3)。此步骤可能需要几秒到几分钟,具体取决于:

- 文件大小
- 计算机性能
- 选择的分辨率

> 💡 **提示**: 当进度条为99%时不要关闭应用程序!

### 下载时出现"Permission denied"

**可能的原因:**

1. 目标文件夹受写保护
2. 同名文件已打开
3. 您的杀毒软件阻止了写入

**解决方案:**

- ✅ 在个人目录(文档、下载)中选择文件夹
- ✅ 关闭任何打开的视频文件
- ✅ 在杀毒软件中添加例外
