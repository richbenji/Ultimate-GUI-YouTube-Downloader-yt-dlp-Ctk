# 🎓 튜토리얼 — GOD (God Offers Downloads)

## 🚀 소개
애플리케이션 사용 가이드에 오신 것을 환영합니다! 이 애플리케이션을 사용하면 YouTube 동영상과 재생 목록을 비디오 또는 오디오 형식으로 다운로드할 수 있으며, 비공개 재생 목록도 지원합니다.

애플리케이션은 두 가지 주요 모드를 제공합니다:

- 🔥 **단순 다운로드**: 동영상을 하나씩 관리(또는 재생 목록)
- 📋 **배치 다운로드**: 여러 URL을 한 번에 다운로드

------

## 🔥 단순 다운로드

1. YouTube 동영상의 **URL을 복사** 🔗

2. URL 필드에 **붙여넣기**

3. "➕ 확인" 버튼 **클릭**

4. 동영상 정보(썸네일, 길이, 크기)가 로드될 때까지 **대기**

   옵션 **선택**:

   - 🎥 **비디오**: 해상도 선택(Best, 2160p, 1440p, 1080p, 720p, 480p, 360p, 240p, 144p)
   - 🎵 **오디오만**: 비트레이트(Best, 320, 256, 192, 128, 96, 64, 32 kbps) 및 형식(M4A 또는 MP3) 선택

   "⬇️ 다운로드" **클릭**

   대상 폴더 **선택**

### 팁 💡

- 다운로드를 시작하기 전에 대기열에 **여러 동영상을 추가**할 수 있습니다
- **전체 크기**와 동영상 수가 다운로드 버튼에 표시됩니다
- **🗑️ 대기열 비우기** 버튼을 사용하여 대기 중인 모든 동영상을 삭제합니다
- 각 동영상 옆의 **ℹ️** 버튼을 클릭하여 세부 정보(사용 가능한 형식, 설명 등)를 확인합니다
- **❌**를 클릭하여 대기열에서 동영상을 제거합니다

## 📚 재생 목록

### 공개 재생 목록

"단일 다운로드" 탭에 재생 목록 URL을 복사하기만 하면 애플리케이션이 모든 동영상을 자동으로 감지합니다! 다운로드를 시작하기 전에 비디오/오디오 옵션을 수정할 수 있습니다.

애플리케이션:

- ✅ 재생 목록의 모든 동영상을 자동으로 로드합니다
- ✅ 각 동영상의 썸네일과 정보를 표시합니다
- ✅ 각 동영상에 대한 옵션을 개별적으로 사용자 지정할 수 있습니다
- ✅ 모든 것을 한 번의 작업으로 다운로드합니다

### 🪪 비공개 재생 목록
#### 방법 1: 브라우저를 통한 자동 연결(권장 ⭐)

**YouTube에 로그인되어 있으면 애플리케이션이 Firefox의 쿠키를 자동으로 사용합니다!**

1. Firefox에서 YouTube 계정에 **로그인**
2. 비공개 재생 목록의 URL을 **복사**
3. 애플리케이션에 **붙여넣기**
4. ✅ **자동으로 작동합니다!** 애플리케이션이 Firefox의 쿠키에 액세스합니다

> 💡 **팁**: 추가 조작 없이 애플리케이션이 항상 비공개 재생 목록에 액세스할 수 있도록 Firefox에서 YouTube에 계속 로그인 상태를 유지하십시오.

#### 방법 2: 쿠키 수동 내보내기(대안)

자동 방법이 작동하지 않거나 다른 브라우저를 사용하는 경우 **YouTube 쿠키**를 제공해야 합니다:

1. 브라우저 확장 프로그램 설치:
   - **Firefox**: [cookies.txt](https://addons.mozilla.org/ko/firefox/addon/cookies-txt/) 또는 [get cookies](https://addons.mozilla.org/ko/firefox/addon/get_cookies/)
   - **Chrome**: [Get cookies.txt LOCALLY](https://chrome.google.com/webstore)

2. YouTube에 로그인
3. Netscape 형식으로 쿠키 **내보내기**(`.txt` 파일)
4. 애플리케이션 왼쪽 상단의 "🪪 ⬆️ cookies.txt" 버튼 **클릭**
5. `cookies.txt` 파일 **선택**
6. 비공개 재생 목록으로 **테스트**

> ⚠️ **경고**: cookies.txt 파일을 공유하지 마십시오. 로그인 자격 증명이 포함되어 있습니다!

### ⚠️ 조언
- cookies.txt 파일을 삭제하지 마십시오
- 오류가 나타나면 쿠키를 다시 로드하십시오

## 📦 배치 다운로드

"배치 다운로드" 탭을 사용하면 모든 동영상에 대해 **동일한 설정**으로 여러 동영상을 다운로드할 수 있습니다.

### 사용법

1. "배치 다운로드" 탭 **열기**
2. 텍스트 영역에 동영상 URL **입력**(한 줄에 하나씩)
   - 또는 "⬆️ 파일에서 로드"를 클릭하여 URL이 포함된 `.txt` 파일 가져오기
3. 유형 **선택**:
   - 🎥 **비디오**: 모두에 대한 공통 해상도
   - 🎵 **오디오만**: 모두에 대한 공통 비트레이트
4. 해상도(비디오인 경우) 및 오디오 비트레이트 **선택**
5. "⬇️ 다운로드" **클릭**
6. 대상 폴더 **선택**

### 단순 다운로드와의 차이점

| 단순 다운로드 | 배치 다운로드 |
| --- | --- |
| 동영상별 **사용자 지정** 옵션 | 모두에 대해 **동일한** 옵션 |
| 썸네일 및 세부 정보 표시 | 단순화된 인터페이스 |
| 다양한 소수의 동영상에 이상적 | 유사한 많은 동영상에 이상적 |

## ❓ 일반적인 문제

### "The playlist does not exist"

**가능한 원인:**

1. 재생 목록이 **비공개**입니다 → Firefox에서 YouTube에 로그인했는지 확인하거나 `cookies.txt` 파일을 제공하십시오
2. URL이 올바르지 않습니다 → 재생 목록의 전체 URL을 복사했는지 확인하십시오
3. 재생 목록이 삭제되었습니다 → YouTube에 여전히 존재하는지 확인하십시오

### "ERROR: unable to download video data"

**가능한 원인:**

1. 불안정한 인터넷 연결
2. 동영상이 삭제되었거나 비공개입니다
3. YouTube가 형식을 변경했습니다 → yt-dlp 업데이트: `pip install -U yt-dlp`

### 느린 다운로드

- **해결책:**
  - ✅ 인터넷 연결 확인 📶
  - ✅ YouTube는 때때로 위치에 따라 속도를 제한합니다
  - ✅ 다른 시간에 다운로드 시도
  - ✅ 더 낮은 해상도로 다운로드(1080p 대신 720p)

### MP3 변환 오류

**오류**: `ERROR: ffmpeg not found`

**해결책**: MP3 변환에는 시스템에 **FFmpeg**가 설치되어 있어야 합니다.

**FFmpeg 설치:**

```bash
# Ubuntu/Debian
sudo apt install ffmpeg

# Fedora
sudo dnf install ffmpeg

# Arch Linux
sudo pacman -S ffmpeg

# macOS (Homebrew)
brew install ffmpeg

# Windows (Chocolatey)
choco install ffmpeg
```

**설치 확인:**

```bash
ffmpeg -version
```

### 다운로드 중 "Permission denied"

**가능한 원인:**

1. 대상 폴더가 쓰기 방지되어 있습니다
2. 같은 이름의 파일이 이미 열려 있습니다
3. 바이러스 백신이 쓰기를 차단하고 있습니다

**해결책:**

- ✅ 개인 디렉토리(문서, 다운로드)의 폴더 선택
- ✅ 열려 있는 비디오 파일 닫기
- ✅ 바이러스 백신에 예외 추가
